# Assets Directory

Place your app assets in this directory:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px or 32x32px)

You can generate these assets automatically using:

```bash
npx expo-cli generate-splash-assets
```

Or use online tools like:
- https://appicon.co
- https://makeappicon.com

For now, the app will work without these assets, but you'll see warnings in the console.
